Resumen como hacer andar:<br />
1-<br />
instalar node.js<br />
Hacer andar un MySQL local (Usando XAMPP u otra forma)<br />

2-<br />
En un command prompt:<br />
Ir al directorio del proyecto (cd "Proyecto-mysql-main")<br />

Usar los siguientes comandos:<br />

node seed.js <br />
(Conecta el MySQL y carga el Fixture)<br />
node server.js<br />
(Conecta el backend y MySQL al host)<br />

3-<br />
Entrar al localhost:3000<br />
