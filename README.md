Resumen como hacer andar:
1-
Hacer andar MySQL local (XAMPP u otro)
instalar node.js

2-
En un command prompt:
Ir al directorio del proyecto (cd "Proyecto-mysql-main")

Usar los siguientes comandos:

node seed.js 
(Conecta el MySQL y carga el Fixture)
node server.js
(Conecta el backend y MySQL al host)

3-
Entrar al localhost:3000
